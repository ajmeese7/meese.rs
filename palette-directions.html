Batch A — restrained directions

Same UI slice, four palettes, one accent each. The first three drop the orange (mono + one cool signal); the last keeps a single warm accent and drops the blue.

// brand · lime family · batch b — neon

Batch B — pushing the lime brighter

The lime, dialed up toward neon. The trick to staying readable: body text never goes lime — it stays neutral near-white. The accent only ever lands on small signal elements (badge, button, link, graph nodes). 04 adds a soft glow to express extra brightness without coloring any text. Note the inline links stay legible at the brighter end.

// brand · neon family · batch c — full spectrum

Batch C — greens, blues & pinks (all with glow)

Eight neon signals across the spectrum, every one carrying the soft glow. Same rule throughout: body copy stays neutral near-white, the accent only touches badge / button / link / nodes. Button label color is tuned per hue for contrast (dark text on the light greens & pinks, white text on the deeper blues & magentas).
