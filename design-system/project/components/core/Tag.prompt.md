Pill-shaped `#tag` label for lower-level keywords; clusters under post cards and drives filters on topic/search pages.

```jsx
<Tag href="/tags/expo-go">expo-go</Tag>
<Tag active onClick={toggle}>eas</Tag>
```

Static `span` by default; becomes an `<a>` when `href` is set. `active` lights the cyan signal for selected filter state. Hash prefix is built in.
