Compact card for short notes that sit inline in the main feed among longer posts; deliberately lighter than `PostCard` (transparent, slate left-rail, log-line rhythm).

```jsx
<NoteCard
  title="Expo Go version mismatch is still a bad failure mode"
  body="SDK/runtime mismatch errors remain more confusing than they need to be."
  date="2026-06-09"
/>
```

`title` is optional — notes can be body-only. Keep the body to a sentence or three.
