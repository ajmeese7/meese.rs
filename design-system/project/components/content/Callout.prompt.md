In-article callout for warnings, notes, and side context inside long-form posts. The house MDX callout — tinted well, left signal rail, mono `// LABEL`, never emoji.

```jsx
<Callout type="warning" title="Watch for SDK/runtime mismatch">
  Expo Go may not support the newest SDK immediately depending on release timing.
</Callout>
```

Types: `note` (cyan), `tip` (green), `warning` (gold), `danger` (red), `context` (violet). `title` is optional.
